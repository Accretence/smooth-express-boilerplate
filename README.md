## Usage

1. Clone this repo:

```bash
git clone https://github.com/Accretence/breathtaking-express-template
```

2. Install the dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Head over to [localhost:5000](http://localhost:5000/api/route) !
